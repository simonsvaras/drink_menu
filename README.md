# Milano Menu

React projekt pro tvorbu editovatelného nápojového lístku pro bar/restauraci Milano.

Cílem projektu je vytvořit menu jako běžnou webovou stránku v Reactu, která má přesný formát A4 a dá se jednoduše exportovat do PDF pro tisk. Díky tomu není potřeba používat nástroje jako Canva nebo Adobe Express a změny v menu lze dělat přímo v kódu.
